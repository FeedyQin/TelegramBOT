## 📖 О проекте

TempleOS Bot — это многофункциональный Telegram-бот с полным набором инструментов для администрирования групп, играми и развлечениями. Все ответы оформлены в стиле [Терри Дэвиса](https://en.wikipedia.org/wiki/Terry_A._Davis) и его операционной системы TempleOS.

Каждая команда работает **с `/` и без**, на **русском и английском**, в **любом регистре** — более 400 синонимов и алиасов.

## ⚙️ Установка и запуск

### 1. Клонируй репозиторий

```bash
git clone https://github.com/your-username/templeos-bot.git
cd templeos-bot
```

### 2. Установи зависимости

```bash
pip install -r requirements.txt
```

### 3. 🔑 Вставь токен бота

Открой файл **`config.py`** и замени строку:

```python
BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"
```

на твой реальный токен, который выдал [@BotFather](https://t.me/BotFather):

```python
BOT_TOKEN = "1234567890:ABCdefGHIjklMNOpqrSTUvwxYZ"
```

> **Где взять токен?**
> 1. Открой Telegram и найди [@BotFather](https://t.me/BotFather)
> 2. Напиши `/newbot`
> 3. Следуй инструкциям — придумай имя и username для бота
> 4. BotFather пришлёт токен — скопируй его в `config.py`

### 4. Запусти бота

```bash
python main.py
```

### 5. Добавь бота в группу и выдай права администратора

Без прав администратора бан/мут/кик и антифлуд не будут работать.

## 🔧 Настройки (`config.py`)

| Параметр | Описание | По умолчанию |
|---|---|---|
| `BOT_TOKEN` | 🔑 Токен от BotFather | `"YOUR_BOT_TOKEN_HERE"` |
| `MAX_WARNS` | Кол-во варнов до автобана | `3` |
| `FLOOD_LIMIT` | Сообщений за `FLOOD_TIME` сек. | `5` |
| `FLOOD_TIME` | Окно антифлуда (секунды) | `10` |
| `ANTIFLOOD_ENABLED` | Включить антифлуд | `True` |
| `ANTISWEAR_ENABLED` | Включить антимат | `True` |
| `BAD_WORDS` | Список запрещённых слов | `[...]` |

## 🛡️ Автомодерация

- **Антифлуд** — при отправке 5+ сообщений за 10 секунд пользователь автоматически мьютится на 5 минут
- **Антимат** — сообщения со словами из `BAD_WORDS` в `config.py` удаляются автоматически
- **Автобан** — при достижении `MAX_WARNS` предупреждений пользователь автоматически банится
- **Статистика** — бот считает сообщения и команды каждого участника

## 🗃️ База данных

Бот использует простую JSON-базу (`data/db.json`), которая создаётся автоматически. Хранит:
- Предупреждения (варны) пользователей
- Данные антифлуда
- Заметки чата
- Статистику участников
- Кастомные приветствия

## 🚀 Деплой на сервер

### Linux / VPS

```bash
# Установи зависимости
pip3 install -r requirements.txt

# Запусти в фоне через screen
screen -S templeos-bot
python3 main.py
# Ctrl+A, D — отключиться от screen

# Или через systemd (рекомендуется):
# создай /etc/systemd/system/templeos-bot.service
```

### systemd сервис (автозапуск)

```ini
[Unit]
Description=TempleOS Telegram Bot
After=network.target

[Service]
User=your_user
WorkingDirectory=/path/to/templeos-bot
ExecStart=/usr/bin/python3 main.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

```bash
sudo systemctl enable templeos-bot
sudo systemctl start templeos-bot
sudo systemctl status templeos-bot
```

### Docker

```dockerfile
FROM python:3.11-slim
WORKDIR /app
COPY requirements.txt .
RUN pip install -r requirements.txt
COPY . .
CMD ["python", "main.py"]
```

```bash
docker build -t templeos-bot .
docker run -d --name templeos-bot --restart always templeos-bot
```
